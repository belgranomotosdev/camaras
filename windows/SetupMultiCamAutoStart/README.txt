
Este paquete configura la ejecución automática del script 'setup-multicam.ps1' al iniciar Windows, como tarea programada en segundo plano para todos los usuarios (cuenta SYSTEM).

1. Extrae todo el contenido del ZIP en una carpeta.
2. Haz clic derecho en `instalar.bat` y selecciona "Ejecutar como administrador".
3. Para desinstalar, ejecuta `desinstalar.bat` como administrador.
